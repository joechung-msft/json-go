# json-go

JSON Parser ported from TypeScript to Go 1.25.0

## License

MIT

## Reference

[json.org](http://json.org)

## Common Go Commands

### Build the project

```sh
go build ./...
```

### Format code

```sh
go fmt ./...
```

### Run the CLI

```sh
go run cmd/cli/main.go
```

### Run tests

```sh
go test ./...
```
